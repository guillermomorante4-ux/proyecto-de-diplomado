# Proyecto de Diplomado

Este repositorio es un espacio colaborativo para organizar y compartir avances, ideas y recursos entre el equipo del diplomado. Aquí encontrarás documentación, ejemplos y herramientas para coordinar reuniones y entregas.

---

## 📌 Objetivos
- Centralizar la información del proyecto.
- Facilitar la colaboración entre los integrantes.
- Documentar procesos, aprendizajes y resultados.
- Coordinar entregas y reuniones de manera organizada.

---

## 📂 Estructura del repositorio
- **/docs** → Documentación oficial, guías y manuales.
- **/examples** → Ejemplos prácticos y casos de estudio.
- **/resources** → Material adicional (lecturas, enlaces, plantillas).
- **/meetings** → Notas y actas de reuniones.

---

## 🚀 Cómo contribuir
1. Haz un *fork* del repositorio.

2. Crea una rama con tu aporte:  
   ```bash
   git checkout -b feature/nueva-idea

3. Realizar tus cambios y subelos:
   git commit -m "Agrego nueva idea"
   git push origin feature/nueva-idea
